Para mi TFG quiero desarrollar un proyecto que combine el mundo del hardware y el software modular, utilizando como ejemplo sistemas montados en una Raspberry Pi con una pantalla genérica conectada a ella, dirigido a un público más casual y no familiarizado con la programación o con conceptos de open source. El proyecto o empresa conceptual se llamara ModevI.

La idea detras de ModevI es crear una marca con un branding atractivo que ofrezca una plataforma tanto para personas especializadas como para usuarios completamente casuales. Esta plataforma se basará en un dispositivo de hardware ligero, pero con conectividad (Bluetooth, Wi-Fi, etc.), presentado como un lienzo en blanco que podrá “pintarse” con cientos de programas.

Para la presentación de ModevI, prepararé solo algunos programas que demuestren la modularidad del software. Estos estarán disponibles en una aplicación o página web, desde la cual se podrán descargar e instalar facilmente, transformando por completo la funcionalidad del dispositivo en cuestión de minutos. tambien se debe poder acceder a la web con las apps desde la misma raspberry pi.
